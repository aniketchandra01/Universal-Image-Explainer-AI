# 🔮 Universal Image Explainer AI

A powerful AI-powered Streamlit application that analyzes any image through multiple lenses — stories, diagrams, notes, concepts, quizzes, mind maps, presentations, and voice explanations.

---

## ✨ Features

### 📌 5 Analysis Modes
| Mode | Description |
|------|-------------|
| 📖 **Story Mode** | Generates a creative narrative from the image |
| 📊 **Diagram Mode** | Explains charts, graphs, and flow diagrams |
| 📝 **Notes Mode** | Converts image content into structured study notes |
| 🧠 **Concept Mode** | Explains like a teacher — deep and insightful |
| 🧒 **ELI10 Mode** | Simplifies complex visuals for a 10-year-old |
